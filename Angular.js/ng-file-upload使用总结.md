一、ng-file-upload简介
　　是一款轻量级、跨浏览器的angular上传文件指令
二、特点
　　（1）支持文件上传进度条、取消、暂停
　　（2）支持文件拖放和黏贴图像
　　（3）支持暂停和取消文件上传
　　（4）支持验证文件的类型 / 大小、图像宽度 / 高度、视频 / 音频持续时间（ng-required）
　　（5）支持预览显示选择的图像、视频、音频
　　（6）支持CORS和直接上传文件的二进制数据( Upload.$http() )
